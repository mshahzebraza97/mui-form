This repository was created to ease the process of creating forms by using **Material-UI** and **Formik**.

# Components

- TextField
- Select (TextField - Single)
- Radio
- Checkbox (Single)
- CheckboxList
- DatePicker
- Field Arrays (Text)
- Field Arrays (Objects)
